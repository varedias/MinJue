# AI 智能设备选择助手

## 功能概述

本项目已集成智谱AI的GLM-4-Flash模型，为用户提供智能设备选型服务。

## 主要功能

### 1. 智能对话助手
- **位置**: `/select-equipment` 页面
- **功能**: 用户可以描述自己的需求，AI助手会提供专业的设备推荐和选型建议
- **支持的设备类型**:
  - AI视觉检测设备
  - 工业相机
  - 镜头光源
  - 测量仪器
  - 工业机器人
  - 包装设备

### 2. 入口位置

#### 商城页面入口
- 路径: `/mall`
- 在搜索栏下方有醒目的AI助手卡片
- 点击"立即体验智能选机"按钮跳转

#### 直接访问
- 直接访问 `/select-equipment` 路由

## 使用流程

1. 用户访问商城或直接进入AI助手页面
2. 点击"开始智能选机"按钮打开对话界面
3. AI助手显示欢迎消息和功能介绍
4. 用户描述需求（例如："我需要检测PCB板缺陷的设备"）
5. AI助手分析需求并提供专业建议：
   - 推荐设备类型
   - 具体型号建议
   - 参数说明
   - 品牌推荐
   - 选型要点

## AI助手特点

- ✅ **专业领域**: 专注于工业设备选型
- ✅ **实时响应**: 快速的对话体验
- ✅ **上下文理解**: 支持多轮对话
- ✅ **品牌推荐**: 了解主流工业设备品牌（海康威视、康耐视、基恩士、Basler等）
- ✅ **参数解释**: 详细解释技术参数和性能指标

## 示例对话

**用户**: 我需要一套检测塑料瓶缺陷的视觉系统

**AI助手**: 根据您的需求，我推荐以下方案：

1. **设备类型**: AI视觉检测系统
2. **推荐配置**:
   - 工业相机: 200-500万像素，GigE接口
   - 光源: 环形或条形LED光源
   - 软件: 支持深度学习的视觉算法
3. **品牌建议**: 海康威视、康耐视、大华智能相机
4. **关键参数**: 
   - 检测精度: 0.1-0.5mm
   - 检测速度: 100-200瓶/分钟
   - 缺陷类型: 裂纹、污渍、变形等

## 注意事项

1. **API Key安全**: 当前API Key直接写在代码中，生产环境建议使用环境变量
2. **错误处理**: 已实现基础错误处理，API失败时会显示友好提示
3. **响应时间**: 通常在2-5秒内得到回复
4. **Token限制**: 单次对话限制1000 tokens
