# AI 智能设备选择助手 - 使用指南

## 🎯 快速开始

### 访问方式

1. **从商城进入**
   - 访问: http://localhost:5173/MinJue/#/mall
   - 在页面上方会看到蓝色渐变的"AI 智能选机助手"卡片
   - 点击"立即体验智能选机"按钮

2. **直接访问**
   - 访问: http://localhost:5173/MinJue/#/select-equipment

## 💬 如何使用

### 第一步：打开AI助手
点击蓝色横幅上的"开始智能选机"按钮，会弹出对话窗口

### 第二步：描述您的需求
在输入框中输入您的设备需求，例如：

#### 示例问题 1: 按应用场景
```
我需要检测PCB板表面缺陷的设备
```

AI会回答：
- 推荐的设备类型（AI视觉检测系统）
- 具体配置要求（相机、光源、软件）
- 推荐品牌（海康威视、康耐视等）
- 技术参数建议

#### 示例问题 2: 设备对比
```
基恩士和海康威视的视觉检测系统有什么区别？
```

#### 示例问题 3: 参数咨询
```
什么是GigE接口？和USB3.0接口有什么区别？
```

#### 示例问题 4: 预算咨询
```
预算3万元，能买到什么样的视觉检测系统？
```

### 第三步：继续对话
- AI助手会记住对话上下文
- 可以继续追问细节
- 可以要求更详细的解释

## 🔧 支持的设备类型

AI助手专注于以下工业设备领域：

| 设备类型 | 说明 | 典型应用 |
|---------|------|---------|
| **AI视觉检测** | 智能缺陷检测系统 | PCB检测、表面质检 |
| **工业相机** | 高精度图像采集 | 机器视觉、检测系统 |
| **镜头光源** | 成像配件 | 视觉系统配套 |
| **测量仪器** | 精密测量设备 | 尺寸测量、三坐标 |
| **工业机器人** | 自动化设备 | 搬运、装配、焊接 |
| **包装设备** | 生产线设备 | 灌装、封口、打包 |

## 💡 提问技巧

### ✅ 好的提问方式

1. **具体场景**
   ```
   我们生产瓶装水，需要检测瓶子是否有裂纹和污渍
   ```

2. **明确参数**
   ```
   需要检测精度0.1mm的视觉系统，预算15万
   ```

3. **对比需求**
   ```
   对比200万像素和500万像素工业相机的区别
   ```

### ❌ 不太好的提问

1. 太宽泛
   ```
   推荐一个设备
   ```

2. 非相关问题
   ```
   如何做营销？（AI专注于设备选型）
   ```

## 🔍 功能特性

### ✨ 主要特点

- **💬 多轮对话**: 支持连续提问，记住对话上下文
- **⚡ 快速响应**: 通常2-5秒内得到专业回复
- **🎯 精准推荐**: 基于GLM-4-Flash模型的智能推荐
- **📚 专业知识**: 了解主流品牌和技术参数
- **🔧 实用建议**: 提供可操作的选型方案

### 🛡️ 安全保障

- 对话内容仅用于设备推荐
- 不存储个人敏感信息
- API调用通过HTTPS加密

## 🆘 常见问题

### Q: AI没有响应怎么办？
A: 
1. 检查网络连接
2. 刷新页面重试
3. 如果问题持续，可以直接浏览设备分类

### Q: AI推荐的设备在哪里购买？
A: 
1. 返回商城浏览具体商品
2. 联系平台上的供应商
3. 使用搜索功能查找推荐的型号

### Q: 可以保存对话记录吗？
A: 当前版本暂不支持保存，建议截图保存重要信息

### Q: AI会犯错吗？
A: AI助手基于训练数据提供建议，但可能不完全准确。重要决策请咨询专业人士。
