## 💡 使用示例

### 示例对话1：新手咨询
```
用户: 我是新手，不知道怎么选择视觉检测设备

AI: 您好！让我为您介绍视觉检测设备的选择要点：

1. **明确应用场景**
   - 检测什么产品？（PCB、塑料件、金属件等）
   - 检测什么缺陷？（尺寸、划痕、颜色等）

2. **关键参数考虑**
   - 检测精度要求（0.01mm、0.1mm等）
   - 检测速度要求（每分钟检测多少个）
   - 工作环境（温度、振动、光照等）

3. **推荐设备组合**
   - 工业相机：200-500万像素
   - 镜头：根据视野和精度选择
   - 光源：环形、条形或背光
   - 软件：视觉处理算法

请告诉我您的具体应用场景，我可以给出更精准的推荐。
```

### 示例对话2：参数咨询
```
用户: GigE接口和USB3.0接口有什么区别？

AI: 这是工业相机常见的两种接口，主要区别如下：

**GigE接口（千兆以太网）**
✅ 优点：
- 传输距离长（可达100米）
- 多相机级联方便
- 工业环境抗干扰强
- 成本相对较低

❌ 缺点：
- 带宽有限（约125MB/s）
- 需要网卡支持

**USB3.0接口**
✅ 优点：
- 带宽大（约350MB/s）
- 即插即用，使用方便
- 适合桌面环境

❌ 缺点：
- 线缆长度限制（5米内）
- 多相机扩展困难

**选择建议**：
- 工业生产线 → 选GigE
- 实验室/桌面应用 → 选USB3.0
- 高分辨率高帧率 → 考虑USB3.0或CameraLink
```
